# Workout Tracker Backend

## Install the project

`npm install`

## Setup the following .env variables

- **PORT**
- **COOKIE_SECRET**
- **JWT_SECRET**
- **MONGO_URI**
- **JWT_VERIFICATION_LIFETIME**
- **LOGIN_LIFETIME**
- **MAILER_USER**
- **MAILER_PASSWORD**
- **TWILIO_SID**
- **TWILIO_AUTH_TOKEN**
- **TWILIO_SMS_PHONE_NUMBER**

## Build the code
`npm run build`

## Start the server
`npm start`

## Start the server in dev mode
`npm run dev`
