# Workout Tracker Backend

## Install the project

_npm install_

## Setup the following .env variables

- **PORT**
- **COOKIE_SECRET**
- **JWT_SECRET**
- **MONGO_URI**
- **JWT_VERIFICATION_LIFETIME**
- **LOGIN_LIFETIME**
- **MAILER_USER**
- **MAILER_PASSWORD**
- **TWILIO_SID**
- **TWILIO_AUTH_TOKEN**
- **TWILIO_SMS_PHONE_NUMBER**
